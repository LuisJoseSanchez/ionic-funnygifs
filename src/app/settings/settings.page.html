Settings Funnygifs

Funnygifs es un cliente de Reddit que muestra los GIFS de las secciones (subreddit) de esta red social. Por defecto, se usa la sección gifs pero puedes elegir cualquier otra sección, por ejemplo perfectloops, chemicalreactiongifs o funny.

Para reproducir un GIF, pulsa sobre él. Para ver el artículo original del que procede el GIF, pulsa el título que aparece justo debajo.

Puedes configurar la aplicación mediante las opciones que tienes aquí debajo.
